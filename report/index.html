项目概况

维度值
语言标准C++20
构建系统CMake 3.20+
产物libchronos.a (静态库)
依赖管理libchronos-deps (FetchContent 聚合)
测试框架GTest + RapidCheck (PBT) + Google Benchmark
测试数量368 tests, 46 suites
CI 矩阵5 Jobs (Ubuntu Debug/Release, macOS Release, ASan+UBSan, TSan)
许可证私有 (源码不可见, libchronos.a 供 trading_engine 链接)

架构分层

层级模块说明
L0 基础core/Decimal (fpm fixed-point), Tick, OrderRequest, Fill, Config, Error
L1 IOio/Transport (WebSocket TLS), Protocol (Binance/OKX JSON), Security (HMAC-SHA256)
L2 工具utils/MPMC Queue (Vyukov 无锁), CPU Affinity
L3 行情market_data/OrderBook V1/V2 (热冷分离), AnyGateway (类型擦除), ExchangeAdapter
L4 交易trading/ + risk/StrategyEngine, RiskEngine (5 项检查 <1μs), PositionManager
L5 执行execution/OrderGateway, BinanceHttpClient, BinanceUserStream
L6 日志logging/LogWriter (二进制), LogReader (mmap), ZMQBridge (PUB-SUB)
L7 回测backtest/TimeReplayer, BacktestEngine, Metrics (Sharpe/Sortino/Calmar)
